Support

This document lists several ways you can find help with using DbxPgSql and describes the DbxPgSql Priority Support program.

Support Options

There are a number of resources for finding help on installing and using DbxPgSql.

  • You can find out more about DbxPgSql installation or licensing by consulting the Readme and FAQ documents.
  • You can get community assistance and technical support on the DbxPgSql Community Forum
  • You can get advanced technical assistance by DbxPgSql developers through the DbxPgSql Priority Support program.

Subscriptions

The DbxPgSql Subscription program is an annual maintenance and support service for DbxPgSql users.

Users with a valid DbxPgSql Subscription get the following benefits:

  • Product support through the DbxPgSql Priority Support program
  • Access to new versions of DbxPgSql when they are released
  • Access to all DbxPgSql updates and bug fixes
  • Notification of new product versions

Priority Support

DbxPgSql Priority Support is an advanced product support service for getting expedited individual assistance with DbxPgSql-related questions from the DbxPgSql developers themselves. Priority Support is carried out over email and has a two business day response policy. Priority Support is available for users with an active DbxPgSql Subscription.

To get help through the DbxPgSql Priority Support program, please send an email to dbxpgsql@devart.com describing the problem you are having. Make sure to include the following information in your message:

  • The version of Delphi or C++Builder you are using.
  • Your DbxPgSql Registration number.
  • Full DbxPgSql edition name and version number. You can find the version number in DLL version information.
  • Versions of the PostgreSQL server and client you are using.
  • A detailed problem description.
  • If possible, a small test project that reproduces the problem. Please include definitions for all objects of other schemas and avoid using third-party components.

If you have any questions regarding licensing or subscriptions, please see the FAQ or contact sales@devart.com
